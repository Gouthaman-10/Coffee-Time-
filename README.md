# ☕ Coffee Time!

**Coffee Time!** is a modern coffee shop website built with **HTML, CSS, and JavaScript**.  
It showcases different coffee varieties through a clean, responsive design and interactive UI.

---

## 🚀 Features
- Responsive design for desktop, tablet, and mobile
- Interactive coffee menu with multiple categories
- Smooth animations and transitions
- Simple, clean, and modern UI
- Built only with HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack
- **HTML5** – Structure and content  
- **CSS3** – Styling and responsiveness  
- **JavaScript (ES6)** – Interactivity and functionality  

---

## 📂 Project Structure
Coffee_Time/
Coffee_Time/
│── index.html → Main homepage (coffee shop layout, menu, sections)
│── style.css → Stylesheet (colors, layout, responsive design)
│── script.js → JavaScript logic (menu interactions, animations)
│── /images/ → Project images (logo, coffee photos, icons)
│── /assets/ → Extra assets (fonts, backgrounds, external files)
